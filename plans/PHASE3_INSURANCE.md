# Phase 3: Insurance & Claims Workflow

## Status: PLANNED

## Objective
Build a complete insurance claims workflow with damage assessment,
before/after comparison, and estimating software integration.

## Planned Features

### 1. Claims Management Dashboard
- Create/manage insurance claims
- Claim lifecycle tracking (opened, inspected, estimated, settled, closed)
- Adjuster assignment
- Claim priority and categorization
- Search and filter claims

### 2. Damage Assessment
- Damage area marking on roof (hail, wind, impact)
- Damage severity classification (minor, moderate, severe, total loss)
- Photo annotation tools
- Test square placement and anomaly counting
- Damage density mapping

### 3. Before/After Comparison
- Dual-pane imagery viewer
- Synchronized pan and zoom
- Side-by-side and overlay modes
- Date-stamped imagery management

### 4. Xactimate Integration
- ESX file format export
- Line item mapping from measurements
- Material and labor cost integration
- Sketch export compatible with Xactimate

### 5. Report Enhancements
- Insurance-specific report templates
- Damage documentation report
- Scope of loss report
- Photo pages with annotations
- Estimate summary

### 6. Disaster Response Mode
- Bulk property assessment
- Triage classification (minor/major/total loss)
- Geographic area damage mapping
- Priority routing for adjusters

## Dependencies
- Claims database schema
- User authentication (Phase 6)
- File upload system
- ESX format specification
