# /build-next - Continue Development to Next Feature

## Description
Reads the continuation prompt and roadmap to identify and implement
the next feature on the development plan.

## Instructions
1. Read CONTINUE.md for the continuation protocol
2. Read ROADMAP.md for current phase and next features
3. Read the relevant phase plan from plans/
4. Use the ARCHITECT agent to plan the implementation
5. Use the BUILDER agent to implement
6. Use the TESTER agent to write tests
7. Use KAREN to verify the work
8. Update all plans, specs, and docs
9. Commit with descriptive message
