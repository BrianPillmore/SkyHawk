# Skill: Generate/Modify Reports

## Description
Add or modify PDF report generation including new report sections,
formatting changes, or new report types.

## Context Files
- src/utils/reportGenerator.ts - PDF generation logic
- src/components/reports/ReportPanel.tsx - Report UI
- specs/REPORT_SPEC.md - Report specification

## Steps
1. Review the report spec for requirements
2. Modify src/utils/reportGenerator.ts for PDF changes
3. Update ReportPanel.tsx for UI changes
4. Update REPORT_SPEC.md if new sections added
5. Test by generating a sample report
